EasyCheck -VATNo

Folosind aplicatia ai urmatoarele beneficii, referitoare la verificarea VATNo-ului partenerilor:

  1. se verifica daca VATNo exista la ANAF (sa nu fie gresit inregistrat in sistem)
  2. se verifica Nume, Adresa, stare generala
  3. status tva la incasare
  4. status inactivi la ANAF
  5. status VATNo in platforma VIES, pentru partenerii din UE
Procesul de verificare este urmatorul:
  1. Prin intermediul aplicatiei QlikView se incarca toate VATNo-urile clientilor si furnizorilor din ERP si se exporta intr-un CSV
  2. Aplicatia QlikView trimite CSV-uri (in pachete de ~250 pcs) in aplicatia WEB, printr-un FTP
  3. Aplicatia web interogheaza in mod automat site-urile ANAF si VIES, dupa un program stabilit de utilizator
  4. Dupa finalizarea interogarii obtinem o baza de date SQL cu informatiile din poza de mai jos.
  5. Informatiile din SQL se incarca intr-o aplicatie QlikView si se pot compara cu informatiile existente in ERP → se marcheaza diferentele privind atributul fiscal, pentru a fi operate corectii in ERP (se genereaza rapoarte de tip alerta)
Interfata de vizualizare: Interfata de vizualizare a datelor este realizata in QlikView si permite utilizatorului sa realizeze filtre rapide dupa diverse criterii.
Pe langa raportul general care prezinta situatia generala a partenerilor (clienti / furnizori), asa cum este preluata din platformele ANAF si VIES, mai pot fi realizate rapoarte specifice de alerta care sa compare informatiile cu starile existente in ERP-ul clientului.
